Settings

        
        
        
        
        
        

        
      

      


        

          
General

          
          
          
Stored locally on this Mac. Bootstrapped from your harness key if present.

          

            
            
          

          

          
Shell sandbox

          
          

          
Permission rules

          
"Always allow" rules — matching non-destructive requests skip the approval prompt.

          

          

          
AI spend

          

          

          
Harness-only, computed from OpenRouter pricing per turn. The account line includes everything on your key.

          

          
Appshots

          
Press ⌘⇧H anywhere on your Mac to capture the screen and attach it to the active chat — Harness comes to the front with the shot ready to send. Requires Screen Recording permission for Harness Code (System Settings → Privacy).

        


        

          
Connectors — MCP servers

          
Stdio MCP servers. Their tools are advertised to every model as mcp__server__tool and are approval-gated like any other tool.

          

          

          
          
          
          

        


        

          
Skills

          
Markdown playbooks in ~/.harness-code/skills/. Invoke one by typing /name in the composer — its instructions are sent to the model with your task.

          

          

          
          
          
          

        


        

          
Plugins

          
A plugin bundles skills and MCP servers in one installable folder: plugin.json ({ name, description, mcpServers: [{name, command}] }) plus skills/*.md. Installed to ~/.harness-code/plugins/.

          

          

          
          
          

        


        

          
Shortcuts

            ⌘KModel picker
            ⌘BToggle sidebar

            ⌘UAttach files/photos
            ⌘⇧HAppshot (global)
            ⌘1–9Switch session
            ⇧TabCycle permission mode
            EscStop the agent / deny
            ⏎Send / approve
          

          

          
Commands

          

            /forkDuplicate this chat (full history) into a new session
            /goalSet a standing goal the model keeps in its system prompt
            /loop/loop <minutes> <prompt> — re-send on an interval · /loop stop
            /compactSummarize & compress the context
            /<skill>Run a skill from ~/.harness-code/skills
